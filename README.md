# Meetup, Client side  

*this is the frontend of a decoupled social networking app designed for seamless user discovery and interaction.*

### Features
- **Discovery Feed:** browse through user profiles with a clean, modern ui.
- **Connection Logic:** send, accept, or reject requests in real time.
- **Secure Auth:** integrated jwt authentication with protected routes.
- **Responsive Design:** built from scratch using daisy ui and tailwindCss to look great on mobile and desktop.

### Tech Stack
- **Library:** react.js
- **Styling:** tailwind Css
- **State Management:** react hooks and redux toolkit
- **Data Fetching:** axios

### Coming Soon..
- **Real time Chat:** implementation using socket.io.
- **Subscription UI:** premium features and payment integration.
